Detailed Description

Application implementation of the delegate for selecting node functionality Also handles clicks to the screen and calls glPickDraw for each scene object If there is a GLDataCenter present in the scene, nodes can be selected by clicking on them The 7 button cycles through each job in the datacenter only showing you nodes that have that job id

Author:
Brock Erwin

Overrides the super implementation of keyPress in order to check for our own special key-presses. After we've checked, we can pass this message to the super class and let it handle the key press however it wants.

Right now, the only thing that this class overrides is the '7' key press.
